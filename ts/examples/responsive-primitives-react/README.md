# Responsive Primitives React Example

## Demonstrates

This example renders the React responsive-control-plane primitives with deterministic tokens and markup. It keeps the shared responsive primitive contracts in the package surface while using the React adapter-specific component entrypoint.

## Run

From `ts/`:

```bash
npm run example:responsive:react
```

The script renders and asserts the expected responsive primitive output.
