# Responsive Primitives Svelte Example

## Demonstrates

This example renders the Svelte responsive-control-plane primitives with the same shared contracts used by the React and Vue examples. It protects Svelte parity for brand-agnostic responsive primitives.

## Run

From `ts/`:

```bash
npm run example:responsive:svelte
```

The script renders and asserts the expected responsive primitive output.
