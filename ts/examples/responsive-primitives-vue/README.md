# Responsive Primitives Vue Example

## Demonstrates

This example renders the Vue responsive-control-plane primitives with the same shared contracts used by the React and Svelte examples. It protects Vue parity for brand-agnostic responsive primitives.

## Run

From `ts/`:

```bash
npm run example:responsive:vue
```

The script renders and asserts the expected responsive primitive output.
